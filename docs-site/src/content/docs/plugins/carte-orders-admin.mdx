`@carte/orders-admin` is the native React admin plugin for Carte v0.1.
It renders order lists and order details, drives status changes, requests
refunds, validates modifier groups, and manages email templates through typed
contracts.

## Primary responsibilities

- Displays snapshotted order items and modifiers exactly as purchased.
- Calls orders-backend routes through typed REST contracts from `@carte/core`.
- Prevents invalid order workflow transitions in the admin UI.
- Keeps the native plugin boundary separate from sandbox-only Worker config.
