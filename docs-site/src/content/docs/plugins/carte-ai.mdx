`@carte/ai` is the commercial native React AI admin plugin for Carte v0.1. It is
read-by-default and only performs writes after an explicit user confirmation,
with before/after diffs, audit entries, and ten-minute undo tokens.

## Primary responsibilities

- Presents AI-suggested operational changes as confirmable diffs.
- Enforces per-turn PII opt-in at the tool-call boundary before LLM egress.
- Records audit entries for AI write actions and allergen changes.
- Provides the interim MCP wrapper Worker for EmDash 0.9.0 deployments.
