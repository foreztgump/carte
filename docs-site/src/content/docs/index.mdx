Carte is the restaurant plugin family for EmDash 0.18: menus, reservations,
Tender-hosted ordering, storefront components, and an AI-native admin panel.

## Plugin family

- [`@carte/core`](./plugins/carte-core/) — menu schema, restaurant settings,
  allergen taxonomy, admin Block Kit, and Schema.org JSON-LD.
- [`@carte/reservations`](./plugins/carte-reservations/) — race-safe
  reservations, holds, confirmation tokens, cancellation, and email.
- [`@carte/orders-backend`](./plugins/carte-orders-backend/) — Tender-hosted
  checkout, idempotent order state, order snapshots, refunds, and email receipts.
- [`@carte/orders-admin`](./plugins/carte-orders-admin/) — native React order
  management UI for status changes, refunds, modifiers, and templates.
- [`@carte/views`](./plugins/carte-views/) — Astro storefront components for
  menus, reservations, carts, checkout, hours, and restaurant details.
- [`@carte/ai`](./plugins/carte-ai/) — AI admin panel with read-by-default
  behavior, write-on-confirm changes, audit entries, and ten-minute undo.

## Launch readiness

Carte runs on **EmDash 0.18** (v0.3.0-rc family). Start with the recipes for the
most common flows:

- [Quickstart](./recipes/quickstart/) — install the plugin family.
- [Menu page](./recipes/menu-page/) — render menus with JSON-LD.
- [Reservation form](./recipes/reservation-form/) — wire the public form.
- [First order](./recipes/first-order/) — Tender-hosted checkout end to end.
- [AI panel](./recipes/ai-panel/) — turn on the AI admin assistant.

The launch gate, ownership, and HITL-deferred items are tracked in
[`LAUNCH_CHECKLIST.md`](https://github.com/foreztgump/carte/blob/main/LAUNCH_CHECKLIST.md).
