Carte is the restaurant plugin family for EmDash: menus, reservations, Stripe
ordering, storefront components, and an AI-native admin panel.

## v0.1 plugin family

- [`@carte/core`](./plugins/carte-core/) — menu schema, restaurant settings,
  allergen taxonomy, admin Block Kit, and Schema.org JSON-LD.
- [`@carte/reservations`](./plugins/carte-reservations/) — race-safe
  reservations, holds, confirmation tokens, cancellation, and email queues.
- [`@carte/orders-backend`](./plugins/carte-orders-backend/) — Stripe Checkout,
  webhook idempotency, order snapshots, refunds, and email receipts.
- [`@carte/orders-admin`](./plugins/carte-orders-admin/) — native React order
  management UI for status changes, refunds, modifiers, and templates.
- [`@carte/views`](./plugins/carte-views/) — Astro storefront components for
  menus, reservations, carts, checkout, hours, and restaurant details.
- [`@carte/ai`](./plugins/carte-ai/) — AI admin panel with read-by-default
  behavior, write-on-confirm changes, audit entries, and ten-minute undo.

## Launch readiness
