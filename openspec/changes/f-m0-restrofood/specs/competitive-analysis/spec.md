## Given

RestroFood vendors menu browsing, WooCommerce ordering, delivery scheduling, branch operations, and admin/frontend templates in one WordPress plugin.

## When

We author `docs/competitive-analysis/restrofood.md` by reading the vendored PHP source and tracing the files that coordinate those domains.

## Then

The teardown must explain how menu, orders, and reservation-like delivery state are shared; how the admin information architecture is organized; how frontend rendering is composed; and which concrete strengths and weaknesses Carte should adopt or reject, all backed by specific `file:line` citations.
